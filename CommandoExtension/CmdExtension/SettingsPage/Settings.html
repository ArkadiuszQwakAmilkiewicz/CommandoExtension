ABOUT

The goal of this extension is to let user use the browser much faster. It includes few tools InSearchBar Comando RightClick Comando PageScritp Comando List Comando

----------------------------------------------------------------------------------------------------------------------

MANUALS

Usage: In a address bar(where the URL is typed) write '@' then hit 'TAB' and send the message to the commando.

----------------------------------------------------------------------------------------------------------------------

Default message build:[Commando's control] + [white_space] + message
Full Message build:[Commando modifire]+ [white_space] + [Commando's control] + [white_space] + message
Modifires:
'-' updates the curent tab
'+' adds new tab
'*' adds new inactive tab

----------------------------------------------------------------------------------------------------------------------

Commando's control can not start with( '-' or '+' or '*' or '!' or '@' or '#' or '/' or '?') or include (white_space)

----------------------------------------------------------------------------------------------------------------------

Right Click Commandos: mark the message >>> right click >>> chose commando

----------------------------------------------------------------------------------------------------------------------

Page script if activated is letting user to quick interactions within a webPage hold ALT to see possible interactions on the page then click CTRL and chose "go to" release the CTRL

Example

Commando Example 1(quick Wiki search)

Start fill: https://en.wikipedia.org/w/index.php?search=
End fill: &title=Special%3ASearch&go=Go&ns0=1
Control: wiki
Description: Wikipedia

Message: wiki WWII
Action: (Redirects to Wikipedia's page about WWII)

Commando Example 2 (quick YouTube search)

Start fill: https://www.youtube.com/results?search_query=
End fill:
Control: yt
Description: YouTube

Message: yt Cute cats
Action: (Redirects to yt page with a search: Cute cats)
